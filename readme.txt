Gumroad Embed Button for WooCommerce allows you to replace the buy button for external/affiliate products with a customizable Gumroad embed button.

== Description ==

This plugin adds a Gumroad embed option for external/affiliate products in WooCommerce. You can customize the button style with options like width, height, font size, and background color.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/gumroad-embed/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Go to the WooCommerce product settings to enable the Gumroad Embed option for external/affiliate products.

== Frequently Asked Questions ==

= How do I customize the button? =

You can change the button's width, height, font size, and background color in the settings page under WooCommerce > Gumroad Embed Settings.

= Does this plugin require WooCommerce? =

Yes, WooCommerce must be installed and activated for this plugin to work.
